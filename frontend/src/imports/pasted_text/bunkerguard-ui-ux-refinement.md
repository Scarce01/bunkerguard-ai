Refine the BunkerGuard UI/UX to feel like a real premium enterprise maritime intelligence platform, NOT an AI-generated dashboard template.

Overall design direction:

* Premium maritime intelligence / cybersecurity / command-center feel
* Inspired by Palantir, Bloomberg Terminal, NATO maritime command systems, modern SOC dashboards
* Dark, cinematic, high-contrast, premium enterprise aesthetic
* More “mission control” and less “generic AI dashboard”
* Reduce clutter and improve visual hierarchy
* Stronger “wow factor” while staying professional and realistic for BDN officers / maritime investigators

==================================================

1. GLOBAL VISUAL SYSTEM REFINEMENT
   ==================================================

Color hierarchy:

Main background:

* Make the overall background significantly darker
* Use deep navy / near-black maritime tones
* Example direction:

  * #03111F
  * #041223
  * #061728
* Avoid flat navy blue

Add subtle intelligence atmosphere:

* Add soft radial gradients
* Very subtle blue glow in background
* No flashy neon cyberpunk

Example feel:
“submarine control room” + “intelligence command center”

Cards:

* Cards should be slightly brighter than the background
* Increase separation between cards and body
* Card background:
  deep blue (#0B213D feel)
* Add subtle blue-tinted shadows
* Add very soft glow edges
* Improve depth perception

Header / top navigation:

* Make header darker than current
* Add glassmorphism effect:

  * darker translucent layer
  * subtle blur
* Add soft bottom border glow
* Make top bar feel premium and layered

Sidebar:

* Keep current layout
* Enhance active state
* Add subtle glow line on active menu
* Add left-edge blue accent
* Stronger visual focus for selected page

Typography:

* Reduce text density
* Larger hierarchy difference:

  * stronger titles
  * smaller secondary text
* Increase whitespace
* Avoid too many tiny labels everywhere
